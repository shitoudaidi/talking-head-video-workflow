Use this file as a routing map. The detailed rules live in focused references so the agent loads only what the current production phase requires.

Numerical defaults are in `../assets/design-system.json`.

Core rule: the speaker is the trust layer, typography is the hierarchy layer, cards are the explanation layer, proof is the credibility layer, and motion is punctuation. Do not let one layer impersonate another.
